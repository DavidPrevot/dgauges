dojox.dgauges.LinearScaler

Object » dojo.Stateful » dojox.dgauges.LinearScaler
dojo.require("dojox.dgauges.LinearScaler");
Defined in dojox.dgauges/LinearScaler.js

The linear scaler. This scaler creates major and minor ticks regularly between a minimum and a maximum. Scalers are responsible for tick generation and various data-transform operations.

Method Summary

  • _buildMajorTickItems()
  • _buildMinorTickItems()
  • computeTicks() Creates or re-creates the ticks for this scaler.
  • constructor()
  • getComputedMajorTickInterval() The computed or user defined major tick interval.
  • getComputedMinorTickInterval() The computed or user defined minor tick interval.
  • positionForValue(value) Transforms a value into a relative position between 0 and 1.
  • valueForPosition(position) Transforms a relative position (between 0 and 1) into a value. value: Number A relative position to transform.
